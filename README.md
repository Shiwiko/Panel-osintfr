- **Discord** : Avatar, Badges, Date de création.
- **Roblox** : UserID, Bio, Date de création.
- **IP Tracker** : Localisation complète & ISP.
- **Email** : Détection de fuites de données.

## 🚀 Installation
1. `pip install -r requirements.txt`
2. `python app.py`
